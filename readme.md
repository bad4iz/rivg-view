# Начальная установка
установка зависимостей
```
composer install
```
- переименуйте `.env.example` в `.env`
- и выполните для генерации `ключа` команду
```
php artisan key:generate
```
# Backbone лежит по 
пока
```
root\public\app
```
- выполните команду для запуска
```
php artisan serve
```

запустите по `http://localhost:8000/` 

Наслаждайтесь !
